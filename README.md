# AI Ask

여러 AI에게 한 번에 물어보는 맥 앱. 두 가지 모드가 있어요.

- **모드 1 · 4개 AI 동시** — 노션 AI·제미나이·챗지피티·클로드 창 4개를 띄우고, 같은 질문을 한 번에 보냅니다.
- **모드 2 · 같은 AI 4개 (독립)** — AI 하나를 골라 그 AI 창만 4개를 띄웁니다. 각 창을 따로 씁니다.

## 사전 준비

- macOS
- Google Chrome 설치
- Node.js 18+ 설치 (https://nodejs.org)
- pnpm 설치: `npm install -g pnpm`

## 설치 (최초 1회)

```bash
git clone https://github.com/orot-ai/ai-ask.git
cd ai-ask
pnpm install
```

## 실행

```bash
pnpm start
```

작은 **버튼 창**이 뜹니다. 여기서 원하는 모드 버튼을 누르면 됩니다.

> 더블클릭으로 쓰고 싶으면 `pnpm dist`로 만든 `dist/` 안의 **AI Ask.app**을 사용하세요.
> 서명되지 않은 앱이라 처음 열 때는 **우클릭 → 열기**로 한 번 허용해야 합니다.

## 최초 로그인

버튼을 눌러 크롬 창이 뜨면 **각 창에서 본인 계정으로 로그인**하세요. 로그인 세션은 내 PC(`~/.ai-ask/profile/`)에만 저장되고, 다음부터는 자동 로그인됩니다. 두 모드가 같은 로그인을 공유합니다.

## 사용법

### 모드 1 · 4개 AI 동시

"4개 AI 띄우기" 버튼 → 노션·제미나이·챗지피티·클로드 창 4개.
아무 곳에서나 `!`로 시작하는 텍스트를 **복사(Cmd+C)** 하면 4개 AI에 동시 전송됩니다.

```
!봄에 대한 시 써줘
!이 에러 메시지 뭔지 설명해줘
```

`!!` 만 복사하면 4개 AI의 마지막 답변을 모아 클립보드에 복사합니다.
`!` 없이 복사하면 아무 일도 안 일어나니 안심하세요.

### 모드 2 · 같은 AI 4개 (독립)

"노션 / 제미나이 / 챗지피티 / 클로드" 버튼 중 하나 → 그 AI 창 4개가 뜹니다.
이 모드는 **자동 전송·수집이 없어요.** 각 창에 직접 입력해 따로따로 씁니다.

## 실행 중에 창 더 추가하기

실행 중에도 버튼이 잠기지 않아요. 다른 버튼을 더 누르면 **같은 세션(로그인 공유)에 창이 4개씩 추가**됩니다.
추가된 창은 살짝 어긋나게 쌓여 뜨고, **독립**이라 `!질문` 자동 전송/수집은 처음 띄운 4개에만 적용돼요.
(로그인 프로필을 공유하므로 "완전히 별개의 두 번째 세션"은 안 되고, 항상 이 한 세션에 창을 더하는 방식이에요.)
"닫기"를 누르면 그 세션의 크롬 창이 모두 정리됩니다.

## 종료

버튼 창의 "종료/닫기"를 누르거나 버튼 창을 닫으면 크롬 창까지 정리됩니다.

## 터미널로만 쓰기 (선택)

GUI 없이 모드 1만 바로 쓰고 싶으면:

```bash
pnpm cli    # = node browser.mjs
```

## 데이터 저장 위치

- `~/.ai-ask/profile/` — 브라우저 프로필 (로그인 세션, 개인 PC에만 저장)

> 로그인 정보는 각자의 PC에만 저장되며 Git에 올라가지 않습니다.
